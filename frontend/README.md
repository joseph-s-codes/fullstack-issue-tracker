# Frontend - Full-Stack Issue Tracker

This is the frontend for the Full-Stack Issue Tracker project inspired by Azure DevOps.

It is currently in early development and focuses on building the UI structure and application flow.

---

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

---

## Getting Started

### Install dependencies
```bash
npm install
```

### Run the app

```bash
npm run dev
```

## Current Status

- Frontend application structure and routing set up
- Login page UI implemented (basic form layout completed)
- Dashboard layout scaffolded with navbar, sidebar, and main content sections
- UI is currently non-functional (no backend integration yet)
- Authentication logic and API connections are still in development

## Notes

This frontend is part of a full-stack issue tracking system inspired by Azure DevOps.

The current focus is on building out the UI structure and preparing the application for backend integration.

Future development will include connecting to a Spring Boot backend for authentication, issue management, and full CRUD functionality.
